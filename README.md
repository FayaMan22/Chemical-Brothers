# Chemical Brothers Detergent Store

## Description
An e-commerce web application for detergent products built with React and Vite.

## Features
- Product listing and detail pages
- Add to cart functionality
- Cart management (update quantity, remove items)
- Checkout flow
- Responsive UI

## Tech Stack
- React
- Vite
- React Router
- Context API

## How to Run
```bash
npm install
npm run dev
